# VK SMM Панель (React + VKUI)

Современная версия фронтенда на React с использованием VKUI компонентов.

## 🚀 Запуск разработки

```bash
npm run dev
```

Приложение будет доступно на http://localhost:3000

## 📦 Сборка для production

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## 🛠 Технологии

- **React 19** - UI библиотека
- **VKUI 8** - компоненты в стиле VK
- **Vite 8** - сборщик
- **React Router** - маршрутизация

## 📁 Структура

```
src/
├── api/
│   └── api.js          # API функции
├── pages/
│   ├── AuthPage.jsx    # Авторизация
│   ├── HomePage.jsx    # Публикация постов
│   ├── GroupsPage.jsx  # Управление группами
│   └── PostsPage.jsx   # Просмотр постов
├── App.jsx             # Главный компонент
└── main.jsx            # Точка входа
```

## 🎨 Особенности

- ✅ Современный дизайн в стиле VK
- ✅ Адаптивная верстка
- ✅ Темная/светлая тема (автоматически)
- ✅ Все функции из Vanilla JS версии
- ✅ Улучшенный UX с VKUI компонентами

## 🔧 Настройка

API запросы проксируются через Vite на `http://localhost:80`

Для изменения настроек отредактируйте `vite.config.js`
