# Changelog: Пользовательские поля

## ✨ Новые возможности

### 1. Новые типы полей

#### **Выбор из списка** (`select`)
- Выпадающий список с заранее заданными вариантами
- Настройка вариантов в админке
- Примеры: Пол, Размер, Возраст, Окрас

#### **Да/Нет** (`boolean`)
- Чекбокс для бинарного выбора
- В посте отображается как "✓ Название" если отмечено
- Примеры: Стерилизован, Привит, Чипирован

### 2. UI для настройки

#### Новый компонент: `PostTypesSettings`
- Визуальный редактор типов объявлений
- Drag & drop для полей (подготовлено)
- Добавление/удаление вариантов для select
- Предпросмотр цветов типов

#### Новый компонент: `GroupSettingsDialog`
- Диалог настройки группы
- Включение/выключение типов объявлений
- Сохранение настроек через API

### 3. Интеграция в таблицу групп
- Кнопка "Настройки группы" в меню действий
- Автообновление после сохранения

## 📝 Миграция

Для существующих групп:
1. Типы объявлений по умолчанию выключены (`enable_post_types: false`)
2. Включите в настройках группы
3. Настройте типы и поля
4. Пользователи увидят новые поля в форме создания поста

## 🐛 Известные ограничения

- Максимум 50 вариантов в select (рекомендуется до 10)
- Нельзя изменить тип поля после создания постов (данные могут потеряться)
- Boolean поля не отображаются в посте, если не отмечены
