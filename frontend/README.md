# Frontend - VK SMM Панель

Фронтенд приложения для управления постами в группах ВКонтакте.

## Структура проекта

```
frontend/
├── index.html              # Главная страница
├── pages/                  # HTML страницы
│   ├── auth.html          # Авторизация VK
│   ├── groups.html        # Управление группами
│   └── posts.html         # Просмотр постов
├── js/                    # JavaScript модули
│   ├── utils.js           # Общие утилиты (API, localStorage)
│   ├── app.js             # Главная страница
│   ├── auth.js            # Авторизация
│   ├── groups.js          # Управление группами
│   └── posts.js           # Просмотр постов
└── css/                   # Стили
    └── style.css          # Основные стили
```

## Функционал

### 1. Авторизация (`pages/auth.html`)
- Авторизация через VK Admin app
- Ручной ввод токена из URL
- Сохранение токена в localStorage

### 2. Главная страница (`index.html`)
- Статус авторизации
- Публикация постов
- Загрузка фото и видео
- Отложенная публикация

### 3. Управление группами (`pages/groups.html`)
- Загрузка списка групп пользователя
- Выбор групп для публикации
- Сохранение выбранных групп

### 4. Просмотр постов (`pages/posts.html`)
- Просмотр постов из своих групп
- Просмотр постов из любой публичной группы
- Фильтры: все, от группы, от пользователей, отложенные, предложенные
- Репост записей
- Копирование постов

## Технологии

- Vanilla JavaScript (без фреймворков)
- CSS3 с градиентами
- Fetch API для работы с backend
- LocalStorage для хранения токенов

## API Endpoints

Все запросы идут через Go backend на порту 80:

- `POST /api/vk/wall/post` - публикация поста
- `POST /api/vk/wall/get` - получение постов
- `POST /api/vk/wall/repost` - репост записи
- `POST /api/vk/wall/copy` - копирование поста
- `POST /api/vk/groups/get` - получение групп
- `POST /api/vk/users/get` - получение пользователя

## Запуск

Фронтенд обслуживается Go backend'ом через встроенный файловый сервер.

```bash
# Запуск через Docker
docker-compose up

# Или локально через скрипт
./run
```

Приложение будет доступно на `http://localhost`
