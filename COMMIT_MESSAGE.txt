feat: добавлены новые типы полей для объявлений (select и boolean)

Добавлены два новых типа пользовательских полей:

1. **Выбор из списка (select)**
   - Выпадающий список с настраиваемыми вариантами
   - Примеры: Пол, Размер, Возраст, Окрас
   - Настройка вариантов в админке

2. **Да/Нет (boolean)**
   - Чекбокс для бинарного выбора
   - Отображается как "✓ Название" в посте VK
   - Примеры: Стерилизован, Привит, Чипирован

Backend изменения:
- Обновлена структура PostTypeField с полем options
- Поддержка новых типов: "select", "boolean"

Frontend изменения:
- Новый компонент PostTypesSettings для настройки полей
- Новый компонент GroupSettingsDialog для настройки группы
- Добавлен Switch компонент (shadcn/ui)
- Интеграция в таблицу групп

Документация:
- docs/POST_TYPES_GUIDE.md - полное руководство
- docs/CUSTOM_FIELDS_QUICK_START.md - быстрый старт
- CHANGELOG_CUSTOM_FIELDS.md - детальный changelog

Обратная совместимость сохранена.
