С технической точки зрения [мини-приложения](https://dev.vk.com/ru/mini-apps/overview) и [игры](https://dev.vk.com/ru/games/overview) ВКонтакте — это веб-приложения. При запуске они открываются внутри UI ВКонтакте. В зависимости от используемой платформы одно и то же приложение может открываться либо в HTML-элементе iframe, либо напрямую ВКонтакте в элементе управления WebView.

```
iframe
```

## https://dev.vk.com/ru/mini-apps/development/iframe-webview#iframeiframe
iframe — это HTML-элемент, который может отображать содержимое других страниц внутри текущей страницы. Этот элемент используется для запуска мини-приложений и игр в десктопной или мобильной версии сайта. При запуске приложения платформа открывает веб-страницу с элементом iframe и загружает клиентскую часть приложения в этот элемент.

```
iframe
```

## https://dev.vk.com/ru/mini-apps/development/iframe-webview#%D0%9F%D1%80%D1%8F%D0%BC%D0%BE%D0%B9%20%D0%B7%D0%B0%D0%BF%D1%83%D1%81%D0%BA%20(%D0%B8%D0%BB%D0%B8%20%D0%B7%D0%B0%D0%BF%D1%83%D1%81%D0%BA%20%D0%B2%20WebView)Прямой запуск (или запуск в WebView)
WebView — специальный элемент управления в Android и iOS, который может отображать веб-страницы. По своей сути он работает как браузер, встроенный в Android- или iOS-приложение. Такие элементы могут называться по-разному в разных операционных системах, но, как правило, их имена содержат подстроку WebView. Поэтому мы используем такое название для обозначения способа в целом.

```
WebView
```

Когда пользователь запускает мини-приложение или игру, мобильное приложение ВКонтакте создаёт внутри себя элемент WebView и загружает в него клиентскую часть мини-приложения или игры. Сам запуск выглядит так, как будто мини-приложение или игра открываются в новом экране и являются частью ВКонтакте. Поэтому пользователи иногда говорят, что приложение открывается напрямую ВКонтакте.

## https://dev.vk.com/ru/mini-apps/development/iframe-webview#%D0%97%D0%B0%D0%BF%D1%83%D1%81%D0%BA%20%D0%BD%D0%B0%20%D1%80%D0%B0%D0%B7%D0%BD%D1%8B%D1%85%20%D0%BF%D0%BB%D0%B0%D1%82%D1%84%D0%BE%D1%80%D0%BC%D0%B0%D1%85Запуск на разных платформах
На разных платформах ВКонтакте может открывать мини-приложения и игры либо в iframe, либо в WebView. На iOS возможны оба способа.

## https://dev.vk.com/ru/mini-apps/development/iframe-webview#%D0%9E%D1%81%D0%BE%D0%B1%D0%B5%D0%BD%D0%BD%D0%BE%D1%81%D1%82%D0%B8%20%D0%B7%D0%B0%D0%BF%D1%83%D1%81%D0%BA%D0%B0%20%D0%BD%D0%B0%20iOSОсобенности запуска на iOS
Если мини-приложение или игра прошли [модерацию](https://dev.vk.com/ru/mini-apps/settings/moderation), то они будут запущены в WebView напрямую ВКонтакте. Для пользователей создание нового элемента происходит незаметно. Внешне приложение выглядит как открытие нового экрана в UI ВКонтакте.
Запуск приложения в WebView в мобильном приложении ВКонтакте для iOS
Если мини-приложение или игра не прошли модерацию, на iOS они будут открыты в мобильной версии сайта ВКонтакте ([m.vk.com](https://vk.com/away.php?to=https%3A%2F%2Fm.vk.com)). При этом пользователю может потребоваться авторизоваться. После запуска будут видны кнопки и другие элементы управления мобильного браузера.
