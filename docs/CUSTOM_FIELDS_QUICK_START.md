# Быстрый старт: Пользовательские поля

## 🎯 Что это?

Пользовательские поля позволяют добавлять дополнительную информацию в объявления о животных.

## 📋 Доступные типы полей:

| Тип | Описание | Пример |
|-----|----------|--------|
| **Строка** | Обычный текст | Кличка: Барсик |
| **Число** | Числовое значение | Возраст: 3 года |
| **Телефон** | Телефон (кликабелен в VK) | +7 (999) 123-45-67 |
| **Ссылка** | URL (кликабелен в VK) | https://shelter.ru |
| **Выбор из списка** ⭐ | Выбор одного варианта | Пол: Мальчик |
| **Да/Нет** ⭐ | Чекбокс | ✓ Стерилизован |

## 🚀 Как настроить за 3 шага:

### Шаг 1: Откройте настройки сообщества в VK Mini App
1. Откройте VK Mini App
2. Перейдите в **Профиль** → **Настройки сообщества**
3. Включите переключатель **Использовать категории и анкеты**

### Шаг 2: Создайте тип объявления
1. Нажмите **+ Добавить категорию**
2. Введите название (например: "Собаки")
3. Выберите цвет (нажмите на цветной кружок)

### Шаг 3: Добавьте поля
1. Нажмите на иконку **списка** рядом с категорией
2. Введите:
   - Название поля (например: "Пол")
   - Переменная (например: "pol")
   - Выберите тип поля
   - Отметьте "Обяз." если поле обязательное
3. Нажмите **Добавить**
4. Для **"Выбор из списка"** добавьте варианты:
   - После создания поля откроется список вариантов
   - Введите текст варианта
   - Нажмите **+ Добавить вариант** для следующего

## 💡 Примеры настройки

### Пример 1: Простое объявление о собаке

**Поля:**
- Кличка (строка, обязательное)
- Пол (выбор из списка: Мальчик, Девочка)
- Возраст (число)
- Стерилизован (да/нет)
- Телефон (телефон, обязательное)

**Результат в VK:**
```
🐕 Барсик ищет дом!

Кличка: Барсик
Пол: Мальчик
Возраст: 3 года
✓ Стерилизован

Контакт: +7 (999) 123-45-67
```

### Пример 2: Объявление о пропаже

**Поля:**
- Вид животного (выбор из списка: Собака, Кошка, Птица)
- Описание (строка, обязательное)
- Был в ошейнике (да/нет)
- Вознаграждение (число)
- Телефон (телефон, обязательное)

**Результат в VK:**
```
🔍 Потерялась собака!

Вид животного: Собака
Описание: Черный лабрадор
✓ Был в ошейнике
Вознаграждение: 5000 ₽

Телефон: +7 (999) 123-45-67
```

## 🎨 Настройка "Выбор из списка"

1. Выберите тип поля **"Выбор из списка"**
2. Нажмите **Добавить вариант**
3. Введите текст (например: "Мальчик")
4. Нажмите **Добавить вариант** снова
5. Введите следующий вариант (например: "Девочка")
6. Повторите для всех нужных вариантов

**Популярные варианты:**

**Пол:**
- Мальчик
- Девочка
- Неизвестно

**Размер:**
- Маленький
- Средний
- Большой

**Возраст:**
- Щенок/Котенок
- Молодой
- Взрослый
- Пожилой

**Окрас:**
- Рыжий
- Черный
- Белый
- Серый
- Пятнистый
- Другой

## ✅ Советы

- Не создавайте слишком много полей (5-8 оптимально)
- Используйте "Выбор из списка" вместо "Строка" где возможно
- Делайте обязательными только критичные поля
- Используйте понятные названия
