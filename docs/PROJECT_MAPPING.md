# Карта Проектов: Текущий vs Старый

Этот документ создан для того, чтобы мы с вами не путались между текущим проектом и старым (`vk-mini-app_full`), из которого переносятся некоторые модули.

---

## 🏗️ ТЕКУЩИЙ ПРОЕКТ (Мы работаем здесь!)
**Путь на вашем ПК:** `/Users/dvinyaninov/smm/` (Репозиторий `vk_zooplaform_post`)

Это наш **основной** проект. Здесь находится актуальный продакшен, бэкенд на Go и новая архитектура для кросспостинга и администрирования.

### Структура папок (Что за что отвечает):
* 📁 **`backend/`** — Бэкенд написан на **Go (Golang)**. Здесь находятся все API-роуты, работа с **удаленной базой данных PostgreSQL**, авторизация VK и планировщик постов.
* 📁 **`vk_app/`** — **Новое VK Mini App** (React + VKUI). Это мини-приложение, которое встраивается в ВКонтакте. Оно общается с нашим Go-бэкендом через `/api/app/`. 
  * **Используемые ключи (Mini App):** `VK_MINI_APP_ID=54560047`, `VK_MINI_APP_SECRET`, `VK_MINI_APP_SERVICE_KEY`.
* 📁 **`frontend/`** — **Веб-панель Администратора** (Vanilla JS + HTML). Интерфейс для настройки групп и ручного управления постами по адресу `vk.zooplatforma.ru`. 
  * **Используемые ключи (Авторизация и Постинг):** 
    * Для серверного API и кросспостинга (Standalone App): `VK_CLIENT_ID=54481712`, `VK_CLIENT_SECRET`, `VK_SERVICE_KEY`.
    * Для обхода блокировок по IP при получении токена на фронтенде используется Implicit Flow через ID официального приложения VK Admin: `client_id=2685278`.
* 📁 **`docs/`** — Документация по текущему проекту (API, развертывание).
* 📄 **`run`** — Основной bash-скрипт для локального запуска всего проекта.
* 📄 **`Dockerfile`** — Файл сборки для деплоя на сервер (включает билд Go и React).

## 🤖 Правило взаимодействия
1. **Разработка и Тестирование**: Локально запускать бессмысленно. Мы пишем код, делаем `git push`, и проверяем изменения сразу на продакшен-сервере.
2. **База данных**: Мы используем **удаленную базу данных PostgreSQL**. При написании кода бэкенда на Go нужно использовать драйвер для Postgres (`pq` или `pgx`), учитывая, что БД крутится на удаленном сервере.
4. **VK API и Токены**: Запрещено использование токенов сообществ (групп) для API ВКонтакте. Все запросы к VK API выполняются исключительно через Standalone токен пользователя-администратора (сохраненный в таблице `vk_accounts`).
5. **Если мы делаем новую фичу**, мы делаем ее в `/Users/dvinyaninov/smm/`.
4. **Если вы говорите: *"Возьми радар из старого проекта"***, я:
   * Иду в `/Users/dvinyaninov/Проекты/vk-mini-app_full/vk_radar_app/...`
   * Читаю старый React-код.
   * Адаптирую его и сохраняю в наш `/Users/dvinyaninov/smm/vk_app/`.
   * При необходимости читаю старый Node.js бэкенд и пишу аналог на Go в `/Users/dvinyaninov/smm/backend/`.
