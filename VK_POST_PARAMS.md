# Параметры публикации постов VK (wall.post)

## Основные параметры

### owner_id (обязательный)
- **Тип**: integer
- **Описание**: ID пользователя или сообщества, на стене которого публикуется запись
- **Формат**: Для групп начинается с `-` (например: `-227624792`)
- **Текущий статус**: ✅ Реализовано

### message
- **Тип**: string
- **Описание**: Текст сообщения
- **Ограничения**: Обязателен, если не указан `attachments`
- **Текущий статус**: ✅ Реализовано

### attachments
- **Тип**: string
- **Описание**: Медиа-вложения (фото, видео, аудио, документы, опросы)
- **Формат**: `{type}{owner_id}_{media_id}` через запятую
- **Примеры**: 
  - `photo-227624792_457239123`
  - `video-227624792_456239456,photo-227624792_457239124`
- **Ограничения**: Максимум 10 медиа (фото + видео)
- **Текущий статус**: ✅ Реализовано (фото и видео)

### from_group
- **Тип**: integer (0 или 1)
- **Описание**: От чьего имени публикуется запись
- **Значения**:
  - `1` - от имени сообщества (по умолчанию)
  - `0` - от имени пользователя
- **Текущий статус**: ✅ Реализовано (всегда 1)

## Дополнительные параметры (не реализованы)

### publish_date
- **Тип**: integer (Unix Timestamp)
- **Описание**: Дата и время отложенной публикации
- **Пример**: `1745395200` (21 апреля 2026, 12:00)
- **Применение**: Планирование постов на будущее
- **Приоритет**: 🔥 Высокий

### signed
- **Тип**: integer (0 или 1)
- **Описание**: Добавить подпись автора к записи
- **Значения**:
  - `1` - добавить подпись (имя пользователя)
  - `0` - не добавлять (по умолчанию)
- **Применение**: Показать, кто из админов опубликовал пост
- **Приоритет**: ⭐ Средний

### mark_as_ads
- **Тип**: integer (0 или 1)
- **Описание**: Отметка "Это реклама"
- **Значения**:
  - `1` - добавить отметку
  - `0` - не добавлять (по умолчанию)
- **Применение**: Обязательно для рекламных постов
- **Приоритет**: ⭐ Средний

### close_comments
- **Тип**: integer (0 или 1)
- **Описание**: Отключить комментарии к записи
- **Значения**:
  - `1` - комментарии отключены
  - `0` - комментарии включены (по умолчанию)
- **Применение**: Для объявлений без обсуждения
- **Приоритет**: ⭐ Средний

### mute_notifications
- **Тип**: integer (0 или 1)
- **Описание**: Отключить уведомления подписчикам
- **Значения**:
  - `1` - уведомления отключены
  - `0` - уведомления включены (по умолчанию)
- **Применение**: Для незначительных обновлений
- **Приоритет**: ⭐ Средний

### copyright
- **Тип**: string
- **Описание**: Ссылка на источник контента
- **Формат**: URL
- **Применение**: Указание авторства/источника
- **Приоритет**: ⭐ Средний

### friends_only
- **Тип**: integer (0 или 1)
- **Описание**: Доступность записи
- **Значения**:
  - `1` - только для друзей
  - `0` - для всех (по умолчанию)
- **Применение**: Ограничение видимости
- **Приоритет**: ⚪ Низкий

### services
- **Тип**: string
- **Описание**: Экспорт в другие соцсети (Twitter, Facebook и т.д.)
- **Применение**: Кросспостинг
- **Приоритет**: ⚪ Низкий

### lat, long, place_id
- **Тип**: float, float, integer
- **Описание**: Геолокация (широта, долгота, ID места)
- **Применение**: Привязка к местоположению
- **Приоритет**: ⚪ Низкий

### guid
- **Тип**: string
- **Описание**: Уникальный идентификатор для предотвращения дублей
- **Применение**: Защита от повторной отправки
- **Приоритет**: ⚪ Низкий

### post_id
- **Тип**: integer
- **Описание**: ID отложенной записи для публикации
- **Применение**: Публикация ранее созданных отложенных постов
- **Приоритет**: ⚪ Низкий

### donut_paid_duration
- **Тип**: integer
- **Описание**: Период эксклюзивного доступа для VK Donut подписчиков
- **Значения**: `-1`, `86400`, `172800`, `259200`, `345600`, `432000`, `518400`, `604800`
- **Применение**: Монетизация контента
- **Приоритет**: ⚪ Низкий

### link_button
- **Тип**: string
- **Описание**: Заголовок кнопки для прикрепленной ссылки
- **Применение**: Кастомизация сниппета ссылки
- **Приоритет**: ⚪ Низкий

### link_image
- **Тип**: string
- **Описание**: ID фото/видео для сниппета ссылки
- **Формат**: `{owner_id}_{media_id}`
- **Применение**: Кастомизация превью ссылки
- **Приоритет**: ⚪ Низкий

## Ограничения VK API

### Контент
- ✅ В посте обязателен один из типов: текст, фото, видео, ссылка
- ✅ Максимум 10 медиа (фото + видео)
- ⚠️ До 10 аудио треков или 1 плейлист (только с фото/видео в карусели)
- ⚠️ Только 1 опрос (не может быть единственным вложением)
- ⚠️ Только 1 файл (документ)

### Форматы
- **GIF**: соотношение сторон от 0.66:1 до 2.5:1
- **Картинки для статьи**: от 1:1 до 1:2, минимум 550px ширина
- **Картинки для ссылки**: от 1:1 до 1:2.5, минимум 500px ширина

## Рекомендации по реализации

### Приоритет 1 (Высокий) - Отложенная публикация
```javascript
// Добавить в форму
<input type="datetime-local" id="publishDate">

// В app.js
const publishDate = document.getElementById('publishDate').value;
if (publishDate) {
    formData.append('publish_date', Math.floor(new Date(publishDate).getTime() / 1000));
}
```

### Приоритет 2 (Средний) - Чекбоксы
```javascript
// Добавить в форму
<input type="checkbox" id="signed"> Подпись автора
<input type="checkbox" id="markAsAds"> Отметка "Реклама"
<input type="checkbox" id="closeComments"> Отключить комментарии
<input type="checkbox" id="muteNotifications"> Без уведомлений

// В app.js
if (document.getElementById('signed').checked) {
    formData.append('signed', '1');
}
```

### Приоритет 3 (Низкий) - Дополнительные поля
```javascript
// Добавить в форму
<input type="url" id="copyright" placeholder="Ссылка на источник">

// В app.js
const copyright = document.getElementById('copyright').value;
if (copyright) {
    formData.append('copyright', copyright);
}
```

## Коды ошибок

- **214** - Access to adding post denied (нет прав на публикацию)
- **219** - Advertisement post was recently added (слишком частые рекламные посты)
- **222** - Hyperlinks are forbidden (ссылки запрещены)
- **13000** - Content upload was denied due to active community strikes (страйк сообщества)
